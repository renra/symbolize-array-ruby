# Symbolize Array

## Installation

    gem install 'symbolize_array'

## Usage

    ['symbolize', 'array', :for_ruby, 2].symbolize! => [:symbolize, :array, :for_ruby, 2]

What to say more? It should not be limited in any way to ruby 2. Enjoy!
